做一页《忠臣死谏》的连环画页面。
画一个皇帝后花园里从"太平"到"死亡"的四个连续瞬间：皇帝与太监宴饮 → 忠臣闯入恸哭直谏 → 太监假哭操控皇帝 → 忠臣被拖走行刑。不是战场，是宫廷。

页面要求：
- 共 4 个 frame
- 版式：必须是 top-wide / middle-two / bottom-wide（T型布局，非2x2网格）
- 适合 mobile 竖屏阅读，纵向滚动时依次看到 P1 → P2 → P3 → P4
- 每个 frame 只画图，不要在图中生成任何文字
- 文字说明放在 frame 下方，由前端单独渲染
- 整页严禁生成 panel 下缘字幕条、漫画 caption 条、整句旁白、整段中文说明

布局结构（必须严格遵守）：
```
┌─────────────────────────────┐
│           Frame 1           │  ← 上方全宽格，后花园宴饮
│    （皇帝与太监推杯换盏）     │
├──────────────┬──────────────┤
│   Frame 2    │   Frame 3    │  ← 中间双格：刘陶恸哭 vs 太监假哭
│  （闯入直谏） │ （跪地假哭）  │
├──────────────┴──────────────┤
│           Frame 4           │  ← 下方全宽格，忠臣被拖走
│     （刘陶大呼被拖走）        │
└─────────────────────────────┘
```
关键约束：
- Frame 1 和 Frame 4 必须是横向宽景，左右占满整个页面宽度
- Frame 2 和 Frame 3 在中间行并排，Frame 2 在左，Frame 3 在右
- 禁止生成2x2等分网格或四宫格布局
- 整体是纵向拉长的T型结构，不是正方形网格

角色视觉锚点（全页必须一致）：
灵帝：
- 年轻皇帝面容，穿龙袍便服（不是朝服，是后花园便宴）
- 表情从慵懒安逸（P1）到愣住（P2）到犹豫心软（P3）
- 不是暴君——是被蒙蔽的昏君

刘陶：
- 中年文臣，面容刚正，穿谏议大夫朝服
- P2 满脸泪痕恸哭，P4 被拖走时张嘴大呼
- 他的眼泪是真的，他的悲痛是真的

十常侍（群体角色）：
- 太监服饰，无需逐个区分
- P1 笑容满面，P2 笑容僵住，P3 摘帽跪地痛哭（假的）
- P3 的哭是操控术——可怜但假的，不是滑稽的

整体画风：
黑白小人书/连环画式速写插图，线条清晰有力，黑白对比强烈。本页是宫廷权谋——皇帝与太监宴饮、忠臣闯入直谏、宦官假哭操控、大臣被拖去行刑。线条要有权谋的阴冷和忠烈的悲壮。不要彩绘，不要海报感，不要摆拍。避免灰度过高，黑就是黑，白就是白。

时空连续性要求（最重要）：
- 四格发生在同一个后花园、同一个时段
- P1 石桌旁宴饮，P2 刘陶闯入同一地点，P3 同一花园太监跪地，P4 回廊向外拖走
- 皇帝衣着跨格一致：龙袍便服
- P1 太平→P2 冲击→P3 操控→P4 死亡，情绪一条线走到底

文字与时代感约束：
- 严禁在画面底部自动加字幕、说明条、对白条、整句解说
- 整页只能使用黑白灰，不允许黄色或任何其他颜色点缀
- 牌匾上的字只能做成模糊的古代竖列墨迹感，不可清晰可读
- 绝对不要现代公告排版

逐格要求：
第 1 格（上方全宽格）：
- 故事功能：setup
- 要画的瞬间：灵帝后花园。皇帝坐石桌旁，身边围着十几个太监，推杯换盏。花开了，太监笑得比花好看。太平。安逸。
- 镜头：中景平拍，皇帝坐上首，太监围坐两侧，石桌有酒菜，花园有假山花木。横向占满画面上半部分。
- 画面提示：皇宫后花园。石桌石凳，皇帝坐在上首——年轻面容，穿龙袍便服，表情慵懒安逸。身边围着十几个太监，穿着宫廷太监服饰，推杯换盏，笑容满面。石桌上有酒壶和几碟菜。背景是假山、花木、回廊。花开得正好。画面太平、安逸——没有人知道天下已经到处在反了。黑白小人书速写，中景平拍，后花园宴饮的闲适感。没有大臣，没有武将，只有皇帝和太监。
- 必须出现：
  - 皇帝与十几个太监在后花园饮宴
  - 氛围安逸，花木繁盛
  - 太监笑容满面
  - 没有大臣在场
- 必须避免：
  - 场面豪华铺张
  - 皇帝表情暴虐
  - 画面中有武将或大臣
  - 穿铠甲
  - 生成竖长条或方形格子（必须是横宽构图）
- 这一格图像必须无字，预留给下方文字说明。

第 2 格（中间左格）：
- 故事功能：pressure（冲击——太平被打破）
- 要画的瞬间：刘陶闯进花园，衣冠不整满脸泪痕，扑通跪在皇帝面前放声大哭。太监们的笑僵在脸上。皇帝愣住了。
- 镜头：中景正面偏右，刘陶跪在前景偏左，皇帝在右上首愣住，背景太监笑容僵住。约占页面宽度一半。
- 画面提示：后花园里。一个人闯了进来——中年文臣，穿谏议大夫朝服，面容刚正，但此刻衣冠不整，满脸泪痕。他扑通一声跪在皇帝面前的地上，仰着头放声大哭。皇帝坐在石桌旁，手里酒杯还端着，整个人愣住了。身后太监笑容僵在脸上，有人手里筷子还举在半空。花园还是那个花园，花还在开，但气氛完全变了。黑白小人书速写，中景构图，闯入者的悲痛打破安逸。
- 必须出现：
  - 刘陶跪地大哭，满脸泪痕
  - 皇帝惊讶愣住
  - 太监笑容凝固
  - 花园还在但气氛变了
- 必须避免：
  - 刘陶表情愤怒（他是悲痛）
  - 太监表情过于夸张
  - 此格占满整个页面宽度
  - 场景变成朝堂大殿
- 这一格图像必须无字，预留给下方文字说明。

第 3 格（中间右格）：
- 故事功能：turn（翻转——假哭打败真话）
- 要画的瞬间：十几个太监齐刷刷摘帽跪成一排，磕头痛哭。哭得比刘陶还大声还伤心。皇帝看着他们面露不忍。假的——但皇帝不知道。
- 镜头：中景侧面，前景太监跪成一排磕头痛哭，背景皇帝坐上面露犹豫不忍，刘陶在画面边缘孤立。约占页面宽度一半。
- 画面提示：后花园石阶前。十几个太监齐刷刷摘了帽子，跪成一排，把头磕在地上。他们在哭——不是真的哭，是精心表演的哭：肩膀耸动，双手举在空中，嘴巴大张，泪水横流。一个太监领头喊着话，表情可怜至极。石阶上方，皇帝坐在那里，看着地上跪着的这群人，表情不是愤怒——是犹豫，是不忍，是心软。画面边缘，站着刘陶——满脸泪痕，但他的真眼泪和身后十几个人的假泪混在一起，分不清了。他被孤立了。黑白小人书速写，中景侧面构图，真假混杂的操控场面。
- 必须出现：
  - 十几个太监摘帽跪成一排
  - 哭相是精心表演的可怜，不是滑稽
  - 皇帝表情犹豫不忍偏信
  - 刘陶在画面边缘孤立
- 必须避免：
  - 太监哭得滑稽可笑
  - 皇帝表情愤怒
  - 刘陶还站着争辩
  - 此格占满整个页面宽度
- 这一格图像必须无字，预留给下方文字说明。

第 4 格（下方全宽格）：
- 故事功能：climax（死亡——真话的代价）
- 要画的瞬间：武士拖着刘陶往外走。刘陶挣扎回头，张嘴大呼。悲壮不是恐惧。身后花还在开。一句话炸在宫墙上。
- 镜头：近景动态，刘陶在画面中心被两个武士架着拖走，回头张嘴大呼。身后花园轮廓模糊。横向占满画面底部。
- 画面提示：皇宫回廊。两个武士架着一个中年文臣——刘陶——往外拖。朝服被扯乱，头发散了，但表情不是恐惧，是悲壮。他挣扎着回过头来，嘴巴张得很大——在喊，用全身力气喊最后一句话。脸上还有泪痕，但此刻眼泪不是悲伤，是看见王朝覆灭却无能为力的绝望。身后远处，后花园轮廓还在——花还在开。黑白小人书速写，近景动态，悲壮不是惨烈。
- 必须出现：
  - 刘陶被武士架着往外拖
  - 回头张嘴大呼的瞬间
  - 表情悲壮决绝，不是恐惧求饶
  - 身后花园花还在开
- 必须避免：
  - 血腥或暴力画面
  - 刘陶表情恐惧或求饶
  - 太过凄惨的行刑场面
  - 生成竖长条或方形格子（必须是横宽构图）
- 这一格图像必须无字，预留给下方文字说明。

布局约束（强制执行）：
- Frame 1 必须是横向宽景，占满页面顶部全宽，高度约30-35%
- Frame 2 必须是中间行左侧，约占半宽，与Frame 3并排
- Frame 3 必须是中间行右侧，约占半宽，与Frame 2并排
- Frame 4 必须是横向宽景，占满页面底部全宽，高度约30-35%
- 整体形成T型纵向结构，不是正方形2x2网格

避免项：
full color, glossy poster, photorealistic rendering, stiff posed figures, rigid composition, over-rendered detail, game splash art, fantasy magic effects, low contrast, gray wash, blurry lines, small text in image

布局避免项（关键）：
2x2 grid layout, four equal panels, quadrant layout, square grid, four panels of same size, vertical strip layout, 2x2 matrix, four-box layout, equal quadrants

特别强调：
- 线条要清晰有力，像刀刻木刻一样
- 黑白对比要强烈，避免中间灰调过多
- P2 刘陶的真哭和 P3 太监的假哭并排——真与假的对比是全页核心张力
- P3 太监的哭必须看起来可怜但不能滑稽——读者知道是假的，皇帝不知道
- P4 是全页最重的画面——忠臣被拖走的悲壮，不是惨烈
- 布局必须是T型（top-wide / middle-two / bottom-wide），不是2x2网格
- 不要生成 panel 下缘解说字幕
- 本页是宫廷场景——后花园、石桌、回廊，不是战场不是大殿
