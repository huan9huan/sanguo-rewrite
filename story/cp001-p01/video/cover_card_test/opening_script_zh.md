# P01 Opening Card Script - 中文测试版

## Card Text

三国演义

第一回 · 桃园结义

第 1 段：乱世序幕

故事从这里开始：朝廷腐败，黄巾四起，一张幽州榜文把刘备推到命运门前。

## Voiceover Option A - 最短版

故事从这里开始。

汉朝末年，天下开始乱了。

黄巾军起兵，幽州贴出招兵榜。

城门前，很快会有人停下脚步。

## Voiceover Option B - 更像视频开场

天下将乱。

黄巾起兵。

幽州城门，贴出一张招兵榜。

故事从这里开始。

## Timing Suggestion

- Opening card hold: 1.5-1.8s
- Voiceover can begin on card and continue into first comic shot
- If the full voiceover is used before motion starts, use Option A only
- If the card overlays the first comic motion, use Option B
