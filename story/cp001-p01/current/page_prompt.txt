做一页《乱世序幕》的连环画页面。
把 p01 做成一页四格连环画，让读者在一页内读清汉朝气数将尽、张角聚众造反、天下大乱的磅礴过程，最后落在幽州出榜招兵的钩子上。

页面要求：
- 共 4 个 frame
- 版式：必须是 top-wide / middle-two / bottom-wide（T型布局，非2x2网格）
- 适合 mobile 竖屏阅读，纵向滚动时依次看到 P1 → P2 → P3 → P4
- 每个 frame 只画图，不要在图中生成任何文字
- 文字说明放在 frame 下方，由前端单独渲染

布局结构（必须严格遵守）：
```
┌─────────────────────────────┐
│           Frame 1           │  ← 上方全宽格，占页面顶部约30-35%高度
│        （宽景构图）          │
├──────────────┬──────────────┤
│   Frame 2    │   Frame 3    │  ← 中间双格，各占约50%宽度，并排
│  （左半宽）   │  （右半宽）   │     总高度约30-35%
├──────────────┴──────────────┤
│           Frame 4           │  ← 下方全宽格，占页面底部约30-35%高度
│        （宽景构图）          │
└─────────────────────────────┘
```
关键约束：
- Frame 1 和 Frame 4 必须是横向宽景，左右占满整个页面宽度
- Frame 2 和 Frame 3 在中间行并排，Frame 2 在左，Frame 3 在右
- 禁止生成2x2等分网格或四宫格布局
- 整体是纵向拉长的T型结构，不是正方形网格

整体画风：
黑白小人书/连环画式速写插图，线条清晰有力，黑白对比强烈。这一页是全景式的乱世画卷，线条要有历史厚重感和刀刻般的力度。大场面需有纵深和气势，小瞬间要有细节和张力。人物群像为主，不做个体肖像。背景从简但要有时代感。不要彩绘，不要海报感，不要摆拍。避免灰度过高，黑就是黑，白就是白。

改编规则：
- 一格只讲一个清楚的历史瞬间
- 第一格龙椅惊蛇是全页氛围起点
- 第三格黄巾洪流是全页视觉高潮
- 所有文字放在 frame 下方的文字区，不放在图里
- 采用黑白小人书/连环画式速写插图
- 大场面用远景，关键瞬间用中景，保持节奏
- 人物群像大于个体细节，这是时代画卷
- 关键视觉锚点要放大、突出、要有辨识度
- 布局必须符合 mobile-first 纵向阅读流

逐格要求：
第 1 格（上方全宽格）：
- 故事功能：setup
- 要画的瞬间：温德殿异象——大青蛇盘踞龙椅，天降警告，帝座空悬。
- 镜头：宽景纵深，殿内龙椅上的蛇与惊散的百官同框。横向占满整个画面上半部分。
- 画面提示：温德殿内，一条大青蛇盘绕在龙椅之上，蛇身缠绕椅背，蛇头高昂。龙椅明显空置，没有皇帝。百官四散奔避，有的跌倒，有的回头张望，动作惊惶。重点是王朝将倾的压抑感。黑白小人书速写，线条有力，殿内纵深构图，龙椅和蛇是视觉中心且占比大，人物动作体现惊惶，背景简洁但要看出朝堂庄严感崩塌。此格必须横向宽景，占满页面顶部宽度。
- 必须出现：
  - 龙椅上盘踞的大青蛇（蛇身盘绕，清晰可见）
  - 明显空悬的帝座（无人端坐）
  - 惊惶散去的朝臣（动作夸张但有节制）
- 必须避免：
  - 做成恐怖片
  - 过度血腥或奇幻感
  - 蛇画得过于抽象或太小
  - 生成竖长条或方形格子（必须是横宽构图）
- 这一格图像必须无字，预留给下方文字说明，不要在画中留对白框。

第 2 格（中间左格）：
- 故事功能：pressure
- 要画的瞬间：张角得天书，信众聚拢，村口门匾上写着"甲子"。
- 镜头：中景，张角居中手持三卷书，身后信众成势，村口门匾醒目。位于中间行左侧，约占页面宽度的一半。
- 画面提示：巨鹿郡村口，一个落第秀才模样的人站在中央，双手捧着三卷竹简（三卷分明，捆扎整齐，体积明显）。身后是从四面聚来的百姓，表情虔诚。背景中村口门楼或牌坊上挂着一块醒目的牌匾，上面大大地写着"甲子"二字。重点是从不起眼的人变成大风暴的酝酿。黑白小人书速写，线条清晰，张角眼神有野心但非漫画式邪恶，信众是真实的穷苦百姓，氛围暗流涌动。"三卷书"和"甲子牌匾"是两个必须突出的视觉锚点。此格位于中间行左侧，与右侧第3格并排。
- 必须出现：
  - 张角（居中，手持书卷）
  - 三卷天书（竹简分三捆，清晰可见）
  - 聚拢的信众（形成人群之势）
  - 村口醒目的"甲子"牌匾（大字，易辨识）
- 必须避免：
  - 南华老仙的奇幻场面
  - 让张角成为正面英雄
  - 书卷画成单卷或模糊不清
  - 甲子标记太小或写在门框上难以辨认
  - 此格占满整个页面宽度（必须是半宽）
- 这一格图像必须无字，预留给下方文字说明，不要在画中留对白框。

第 3 格（中间右格）：
- 故事功能：turn
- 要画的瞬间：星夜举兵，数十万人裹黄巾，天公将军大旗高扬。
- 镜头：远景转中景，黄巾洪流如潮，张角立于高台显眼处。位于中间行右侧，约占页面宽度的一半。
- 画面提示：夜色中，数十万人裹着黄头巾，举着火把和农具，从四面八方涌出。张角立于一处明显的高台或土坡上，身后是一面巨大的、独特的战旗——旗面宽大，旗杆高耸，旗上有一醒目的星形或圆形符号代表"天公"（避免细小文字）。重点是民心崩塌、天下大乱的磅礴场面。黑白小人书速写，全景式构图，人潮如洪流向前涌，火光映照面庞，气势重于细节。画面要有压迫感和不可阻挡的力量感。"张角在高处"和"巨大战旗"是视觉锚点。此格位于中间行右侧，与左侧第2格并排。
- 必须出现：
  - 黄巾信众人潮（全景，人数众多）
  - 火把与夜色（明暗对比强烈）
  - 天公将军大旗（旗面宽大，有独特符号，无需文字可辨识）
  - 张角立于高处（位置突出，可见人影）
- 必须避免：
  - 把起义画成正义运动
  - 详细战斗场面
  - 美化张角
  - 大旗上写细小文字（应使用符号替代）
  - 此格占满整个页面宽度（必须是半宽）
- 这一格图像必须无字，预留给下方文字说明，不要在画中留对白框。

第 4 格（下方全宽格）：
- 故事功能：hook
- 要画的瞬间：幽州城门出榜招兵——榜文贴出，无人知谁会来。
- 镜头：收束中景，城门口榜文与来往百姓。横向占满整个页面底部宽度。
- 画面提示：幽州城门口，一张招兵榜文刚贴上城墙，榜文面积较大，位于画面视觉中心偏左或偏右的黄金位置。几个百姓驻足看榜，手指指向榜文，更多人挑着担子匆匆走过。榜文是这一格的绝对视觉锚点，必须足够大、位置足够突出。画面安静但有张力，日常场景里藏着即将到来的波澜。黑白小人书收束镜头，线条清晰，对比度强。此格必须横向宽景，占满页面底部宽度。
- 必须出现：
  - 城门口的招兵榜文（面积大，位置突出，视觉锚点）
  - 来往百姓（有驻足看的，有走过的）
  - 城墙/城门背景
- 必须避免：
  - 提前暗示刘备
  - 榜文上写太多字或画成复杂图案
  - 榜文太小或被遮挡
  - 拖慢收束节奏
  - 生成竖长条或方形格子（必须是横宽构图）
- 这一格图像必须无字，预留给下方文字说明，不要在画中留对白框。

布局约束（强制执行）：
- Frame 1 必须是横向宽景，占满页面顶部全宽，高度约30-35%
- Frame 2 必须是中间行左侧，约占半宽，与Frame 3并排
- Frame 3 必须是中间行右侧，约占半宽，与Frame 2并排
- Frame 4 必须是横向宽景，占满页面底部全宽，高度约30-35%
- 整体形成T型纵向结构，不是正方形2x2网格
- Mobile竖屏阅读时应该是：看到P1 → 往下看到P2和P3并排 → 再往下看到P4

避免项：
full color, glossy poster, photorealistic rendering, stiff posed figures, rigid composition, over-rendered detail, game splash art, palace luxury, infographic collage, fantasy magic effects, horror movie gore, low contrast, gray wash, blurry lines, small text in image

布局避免项（关键）：
2x2 grid layout, four equal panels, quadrant layout, square grid, four panels of same size, vertical strip layout, 2x2 matrix, four-box layout, equal quadrants

特别强调：
- 线条要清晰有力，像刀刻木刻一样
- 黑白对比要强烈，避免中间灰调过多
- 关键视觉锚点要放大、突出、有辨识度
- 文字类信息用符号、形状、大小来替代，不要依赖细小文字
- 布局必须是T型（top-wide / middle-two / bottom-wide），不是2x2网格
