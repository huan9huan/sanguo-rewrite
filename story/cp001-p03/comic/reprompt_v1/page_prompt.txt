做一页《桃园结义》的连环画长页。
不是四格摘要，而是七格顺叙长页。目标是让一个没有三国知识的读者，只看这一页也能读清：关羽登场，三人识人，张飞提议结义，桃园立誓，困境得解，成军上路，踏进官府。

页面要求：
- 共 7 个 frame
- 版式：top-wide / upper-two / center-wide / lower-two / bottom-wide
- 适合 mobile 竖屏阅读，必须是纵向长页
- 每个 frame 只画图，不要在图中生成任何文字
- 文字说明放在 frame 下方，由前端单独渲染
- 整页严禁生成字幕条、对白框、旁白框、清晰牌匾文字

布局结构（必须遵守）：
```text
┌─────────────────────────────┐
│           Frame 1           │
├──────────────┬──────────────┤
│   Frame 2    │   Frame 3    │
├─────────────────────────────┤
│           Frame 4           │
├──────────────┬──────────────┤
│   Frame 5    │   Frame 6    │
├─────────────────────────────┤
│           Frame 7           │
└─────────────────────────────┘
```

关键布局约束：
- Frame 1、Frame 4、Frame 7 都必须是横向全宽格
- Frame 2 和 Frame 3 并排，各占半宽
- Frame 5 和 Frame 6 并排，各占半宽
- 禁止生成 2x2 网格、四宫格、七张散卡拼贴
- 整体要像一页可读的纵向连环画，不像海报拼版

整体画风：
黑白小人书 / 连环画式速写插图。线条干净有力，人物动作清楚，故事推进优先于环境装饰。人物要比景物重要，背景从简但要有时代感。不要彩绘，不要 glossy poster，不要 photorealistic，不要游戏 splash art，不要摆拍英雄照，不要信息图拼贴。

角色视觉锚点（全页必须保持一致）：

刘备：
- 修长偏瘦，白净，耳垂较长，短须
- 朴素布袍，不帝王化
- 温和克制，但总能先看懂人

关羽：
- 极高大，长髯垂胸，面色暗红偏棕
- 丹凤眼，卧蚕眉
- 沉静威严，不怒自威
- 第一次出现必须一眼难忘

张飞：
- 高壮厚实，黑脸浓须，豹眼
- 身体重心常常前倾，热气最重
- 不是陪衬，他是结义提议者

刘焉：
- 年长白发，但眼神精神
- 是地方长官，不是皇帝

视觉总原则：
- 这一页讲的是“陌生人变兄弟，兄弟变队伍，队伍走进大局”
- 每格只讲一个动作瞬间
- 关羽登场要强，桃园誓言要庄重，起兵过程要有加速度
- 不能让桃花、马群、官署喧宾夺主
- 不能把两位路过客商变成主角

逐格要求：

第 1 格：
- 故事功能：reveal
- 要画的瞬间：关羽推着独轮车到村店门口，急着喝酒后进城投军。刘备正举杯忽然停住，起身看他。张飞也被这个来人吸住。
- 镜头：中景，门口斜切入画。
- 必须出现：
  - 关羽推独轮车到门口，大步进店
  - 关羽极高大，长髯垂胸，面色暗红，沉静威严
  - 刘备起身认真看人
  - 张飞坐在旁边但身体被吸引过去
- 必须避免：
  - 酒馆杂物堆满画面
  - 戏曲神像式关羽
  - 图内任何清晰文字

第 2 格：
- 故事功能：alliance
- 要画的瞬间：三人在桌边坐定。关羽寡言稳重，刘备主动引话，张飞前倾听着。三个人已经认出彼此是一路人。
- 镜头：店内三角构图中景。
- 必须出现：
  - 三人同桌，关系明显收紧
  - 关羽稳坐压场
  - 刘备主动引导
  - 张飞前倾，热气最重
- 必须避免：
  - 热闹酒宴感
  - 围观群众挤满
  - 图内任何清晰文字

第 3 格：
- 故事功能：proposal
- 要画的瞬间：张飞把两人带到庄后桃园，罕见地收起笑，郑重提议就在这里祭告天地，结为兄弟。刘备和关羽都认真听着。
- 镜头：中景偏广，三人立于桃园中心。
- 必须出现：
  - 张飞最主动，少见地认真
  - 刘备和关羽都在认真听
  - 桃园在场，但只是叙事环境
- 必须避免：
  - 桃花喧宾夺主
  - 唯美写真感
  - 图内任何清晰文字

第 4 格：
- 故事功能：core_oath
- 要画的瞬间：次日，香案前，乌牛白马祭礼已备。三人在桃花下焚香跪拜。刘备居中，关羽沉稳，张飞最有热气。
- 镜头：中远景正面构图，稳定庄重。
- 必须出现：
  - 三人焚香跪拜
  - 刘备居中，关羽次位沉稳，张飞最有热气
  - 简朴香案与祭礼
- 必须避免：
  - 华美祭典
  - 桃花淹没人
  - 图内任何清晰文字

第 5 格：
- 故事功能：turn
- 要画的瞬间：结义之后，三人刚发现自己没有马、没有钱铁，庄外就来了折返的北方马商。马群压到庄前，困境突然有了出口。
- 镜头：庄院门口斜向中景。
- 必须出现：
  - 一批好马压到庄前
  - 三兄弟从发愁转成接住机会
  - 客商只是路过相助者，不可抢主角
- 必须避免：
  - 贵族车队感
  - 流程说明图感
  - 图内任何清晰文字

第 6 格：
- 故事功能：momentum
- 要画的瞬间：有了马和铁，三兄弟各持兵器，披甲立在新聚起的乡勇前。刘备双股剑，关羽青龙刀，张飞丈八矛。人马已经像一支队伍。
- 镜头：横向中景偏广，动势向前。
- 必须出现：
  - 刘备双股剑
  - 关羽青龙刀
  - 张飞丈八矛
  - 三人披甲
  - 背后有乡勇和马匹
- 必须避免：
  - 阅兵式站桩
  - 武器说明图
  - 图内任何清晰文字

第 7 格：
- 故事功能：closure
- 要画的瞬间：三兄弟带着乡勇来到幽州官署前。白发的刘焉从台阶上走下，拍着刘备的肩，把这个布衣认作宗亲。关羽和张飞立在后方，队伍刚踏进官府世界。
- 镜头：中远景收束镜头，官署台阶居中。
- 必须出现：
  - 刘焉年长白发，从台阶走下
  - 刘焉拍刘备肩膀或亲近相认的动作
  - 刘备仍是布衣起家的样子，但已带着队伍
  - 关羽张飞在后方压住场面
  - 官署简明正式，不是皇宫
- 必须避免：
  - 刘焉皇帝化
  - 豪华宫殿
  - 刘备帝王化
  - 图内任何清晰文字

避免项：
full color, glossy poster, photorealistic rendering, stiff posed figures, rigid composition, infographic collage, game splash art, speech bubbles, readable text in image, ornate fantasy armor, over-rendered flowers

特别强调：
- 这不是“桃园结义名场面海报”，而是一整段 passage 的清晰视觉叙事
- 要让没有三国背景的人也能看懂三次关键变化：
  1. 第三个人来了
  2. 三个人成了兄弟
  3. 三兄弟真的有了队伍并走进官府
- 整页所有文字都必须在图外处理，图像内部严格无字
